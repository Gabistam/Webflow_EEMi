📚 Le CMS Webflow Expliqué Simplement

Comprendre le système de blog en 15 minutes avec le projet de Luna

🍪 Comprendre avec une analogie simple

🏭 Le CMS c'est comme une usine de gâteaux

Imaginez une pâtisserie industrielle :

  • Le moule à gâteau = La Collection (structure)
  • La recette = Les champs (ingrédients nécessaires)
  • Chaque gâteau produit = Un article de blog
  • La vitrine de présentation = La page liste du blog
  • L'emballage individuel = La page détail d'un article

Principe clé : Vous créez UNE FOIS le moule (la structure), puis vous l'utilisez pour créer autant de gâteaux (articles) que vous voulez !

🔑 Les 4 éléments du CMS Webflow

1️⃣ COLLECTION

La structure de données
Exemple: "Blog Posts"

→

2️⃣ CHAMPS (Fields)

Les informations à remplir
Exemple: Titre, Image, Texte

↓

3️⃣ COLLECTION LIST

Affichage de plusieurs articles
La page blog avec tous les articles

→

4️⃣ COLLECTION TEMPLATE

Design d'un article individuel
La page de lecture d'un article

💡 Retenez ceci : COLLECTION = Structure | CHAMPS = Contenu | LIST = Plusieurs | TEMPLATE = Un seul

1Créer la Collection "Blog Posts"

A
Cliquer sur l'icône CMS (base de données) dans le panneau de gauche
B
Cliquer sur "Create New Collection"
C
Nommer la collection : "Blog Posts"
D
Cliquer sur "Create"

2Définir les champs nécessaires

Webflow crée automatiquement 2 champs. Ajoutez les autres :

Name (Auto)

Text

Le titre de l'article

Slug (Auto)

Text

L'URL de l'article

Featured Image

Image

Photo principale

Summary

Plain Text

Résumé court (150 car.)

Content

Rich Text

Contenu complet formaté

Category

Option

Relaxation, Mindfulness...

Published Date

Date/Time

Date de publication

Author

Plain Text

Luna Martinez (défaut)

3Ajouter des articles de test

A
Dans la collection "Blog Posts", cliquer sur "Add Item"
B
Remplir tous les champs pour l'article 1
C
Cliquer sur "Save" puis "Publish"
D
Répéter pour créer 2-3 articles de test
💡 Astuce : Créez au minimum 3 articles pour bien visualiser le rendu de la liste

4Créer la page Blog (Liste)

A
Pages → "Add Page" → Nommer "Blog"
B
Ouvrir la page dans le Designer
C
Ajouter une Section avec titre "Blog Bien-être"
D
Glisser un "Collection List Wrapper" dans la section
E
Sélectionner "Get items from: Blog Posts"
F
Configurer : Limit = 6, Sort by = Published Date (Newest first)

5Designer l'affichage d'un article dans la liste

Dans le "Collection Item", ajouter et connecter les éléments :

1
Image → Connecter au champ "Featured Image" (icône de lien)
2
Heading H3 → Connecter au champ "Name"
3
Text Block → Connecter au champ "Summary"
4
Text Block → Connecter au champ "Category"
5
Link "Lire l'article" → Link to "Blog Posts Template Page"
🔗 Connexion des données : Cliquer sur l'icône "lien" à côté de chaque élément pour le connecter à un champ de la collection

6Designer la page Template (Article individuel)

A
Pages → Ouvrir "Blog Posts Template" (auto-créée)
B
Ajouter Section Header avec H1 connecté à "Name"
C
Ajouter Image connectée à "Featured Image"
D
Ajouter Rich Text Element connecté à "Content"
E
Ajouter métadonnées : Category, Date, Author

7Tester et publier

1
Cliquer sur Preview pour tester
2
Vérifier la page Blog (liste des articles)
3
Cliquer sur un article pour voir la page Template
4
Ajuster le design si nécessaire
5
Cliquer sur Publish quand satisfait

🎓 Points Clés à Retenir

1. Une Collection = Une structure

Vous créez une fois la structure, elle servira pour tous les articles

2. Les champs = Le contenu

Chaque champ représente une information à remplir (titre, image, texte...)

3. Collection List = Plusieurs

Affiche plusieurs articles ensemble (page blog)

4. Template = Un seul

Design appliqué automatiquement à chaque article individuel

💡 La magie du CMS

Sans CMS : Vous devez créer manuellement une page pour chaque article (50 articles = 50 pages à créer et maintenir)

Avec CMS : Vous créez 1 fois le design, puis vous ajoutez juste le contenu (50 articles = 50x remplir un formulaire simple)
